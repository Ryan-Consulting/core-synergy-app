# Core Synergy - Reformer Pilates Booking App

A professional booking and membership management app for Core Synergy Pilates Studio in Milperra, NSW.

## Features

- **Class Scheduling & Booking** - Browse and book pilates classes
- **Membership Management** - Choose from multiple membership tiers
- **Class Packs** - Purchase flexible class packages
- **Shopping Cart** - Review and purchase items
- **Responsive Design** - Works perfectly on mobile and desktop

## Deployment to Vercel

### Option 1: Deploy via Vercel Dashboard (Easiest)

1. Go to https://vercel.com
2. Click "Add New" â†’ "Project"
3. Click "Import Git Repository"
4. Upload this folder or connect your GitHub repository
5. Vercel will auto-detect Next.js settings
6. Click "Deploy"
7. Your app will be live in 2-3 minutes!

### Option 2: Deploy via Vercel CLI

1. Install Vercel CLI: `npm i -g vercel`
2. Navigate to this folder in terminal
3. Run: `vercel`
4. Follow the prompts
5. Done!

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the app.
